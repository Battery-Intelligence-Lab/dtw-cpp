# Method

DTW-C++ clusters time-series data using [dynamic time warping](dtw.md) to compute distance metrics, and [mixed integer programming](mip.md) or [k-Medoids](k-medoids.md) for clustering. The fundamentals of each of these techniques are described in this methods section. Additionally, users can read our [journal article](../publications/joss-paper.md) for more information and relevant references.

For details on the available [clustering algorithms](algorithms.md) (Lloyd iteration, FastPAM, CLARA) and [distance metrics](metrics.md) (L1, L2, Squared L2), see the dedicated pages.
