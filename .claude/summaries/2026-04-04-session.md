- **`dtwc/core/matrix_io.hpp`** (NEW): Free functions in `dtwc::io` namespace:

## Key Decisions

- `operator<<` lives in `dtwc::core` (not `dtwc::io`) so ADL finds it for `DenseDistanceMatrix` arguments.
- `matrix_io.hpp` is added to `dtwc/dtwc.hpp` so existing callers including the main header get it automatically.
- No `std::deque` anywhere — always contiguous ring buffers with head/tail indices.
- ADTW pruned strategy: LB_Keogh is a valid lower bound proved as `LB_Keogh ≤ DTW ≤ ADTW(penalty≥0)`.
